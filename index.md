### **1. The app is not loading new questions.**
The app loads fresh questions from a GitHub repository (if available).  
If you see only old questions:
- Ensure internet connectivity  
- Try reopening the app  
- Try clearing cached data (reinstall the app)

---

### **2. I encountered a blank screen or crash.**
Try the following steps:
- Close and reopen the app  
- Restart your device  
- Ensure you are using the latest version of the app  
- Reinstall the app if issue persists  

If the problem continues, please report it using the contact options below.

---

### **3. My progress is not saved.**
ReactGauge stores progress locally within your device.  
If you uninstall the app, your local quiz history will be removed.
